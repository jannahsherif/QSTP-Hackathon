# STARS — AI Mission Control for Social Media

A futuristic SaaS platform for **Stars of Science** — an AI social media co-pilot with mission control aesthetics, glassmorphism UI, and interactive data visualizations.

## Features

- **Audience Digital Twin** — Three.js interactive globe with audience clusters
- **Social Media Flight Simulator** — Predict performance across 5 platforms
- **Viral DNA Analyzer** — Animated DNA visualization scoring 6 content dimensions
- **Algorithm Decoder** — Platform-specific algorithm insights
- **Mission Control Dashboard** — Real-time analytics center
- **AI Campaign Copilot** — Conversational marketing assistant

## Tech Stack

- Next.js 15 · React 19 · TypeScript
- Tailwind CSS 4 · Framer Motion
- Three.js · React Three Fiber · Drei

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to enter Mission Control.

## Build

```bash
npm run build
npm start
```
